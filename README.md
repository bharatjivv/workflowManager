# 🧑‍💼 Team Task Manager (Full-Stack)

A full-stack web application that allows teams to manage projects, assign tasks, and track progress with role-based access control (Admin/Member).

---

## 📌 Overview

The Team Task Manager helps teams collaborate efficiently by providing:

- Project management
- Task assignment and tracking
- Role-based access control
- Progress monitoring via dashboard

## 🧩 Features

### 🔐 Authentication
- User Signup & Login
- Secure password hashing
- JWT-based authentication

### 👥 Role-Based Access
- **Admin**
  - Create/manage projects
  - Assign tasks
  - Manage team members
- **Member**
  - View assigned projects
  - Update task status

### 📁 Project Management
- Create, update, delete projects
- Assign team members
- Track progress

### ✅ Task Management
- Create and assign tasks
- Status tracking:
  - To Do
  - In Progress
  - Completed
- Due date support

### 📊 Dashboard
- Total tasks
- Completed tasks
- Pending tasks
- Overdue tasks
